Alfred安装快速新建文件扩展

Posted on 2018-10-03 | In MacOS | Comments:

用了2年多的Mac才发现在访达里没有新建文件的按钮，这和Windows相比确实有些不适。不过用了这么久我一般都是使用的终端来进行的操作。不得不说在MacOS上iTerm2这款终端软件简直就是一个神器了。

最近国庆在家瞎折腾的时候就有了这个需求，所以在网上搜索了一下发现了好几款类似的扩展插件。但是用了几个以后发现还是Create New File Extension这款用着比较顺手吧。

安装

在安装之前你必须购买Alfred的授权，也就是要激活Powerpack。然后下载扩展。等待下载完成后直接双击就可以导入了。基本上没什么操作可言，直接点击左下角的import即可。

操作

激活Alfred以后直接输入new就可以了，如果你直接回车那么会默认生成一个叫untitled.txt的文件。所以一般我都是直接new index.php来创建一个叫index的php类型文件。

注意：在新建文件之前，你最好打开访达界面，定位到你要新建文件的那个目录。然后默认是新建到桌面的。比如，我在QQ聊天界面呼出Alfred，然后我执行new lepig.go，那么这个时候会生成一个lepig.go文件存放到桌面上。
