正文

在购买了斐讯K3后，第一件事就是刷机，以下是具体操作步骤，建议全程处于 chrome 浏览器的“无痕窗口”下进行。

第一部分

官方固件降级 58 固件，降级 246 固件

输入路由器地址 p.to 或者 192.168.2.1 进入路由器后台页面，找到“功能设置”，点击“系统状态”

然后查看“硬件版本”和“软件版本”我这边的“硬件版本”是“A2”，“软件版本”是 V21.6.12.6

我们需要将软件版本“V21.6.12.6”先降级为“V21.6.11.58”，在“功能设置”选项中往下滑，找到“手动升级”选项

选择固件“K3_V21.6.11.58p.bin”，然后点击“上传升级”

然后会提示“正在升级”，K3的屏幕上面会显示“系统升级中”

升级完成会自动刷新网页，重新进入“手动升级”页面。，可以看到当前版本为“V21.6.11.58”，然后我们还需要再次降级，降级为“V21.5.37.246”我们再选择“升级文件”，选择“K3_V21.5.37.246.bin”，然后点击“上传升级”，然后会再次进入如上图一样的“正在升级”页面。

升级成功后，登录K3，我们打开“功能设置”中的“系统状态”，可以看到固件版本已经降级为了“V21.5.37.246”，然后执行一次重启路由器操作，确保接下来的操作不出问题。

第二部分

激活 telnet 端口，备份K3

激活 telnet 工具地址：http://www.right.com.cn/forum/thread-261028-1-1.html

打开 RoutAckProV1B2.exe，“路由器 IP”默认为 192.168.2.1，斐讯默认的路由器地址，不需要更改，然后先点击“打开 Telnet”，然后点击“设置永久标志”，输出信息如下“收到应答信号，打开 Telnet 完成，收到应答信号，设置永久标志完成”，最后关掉软件即可

插入 U 盘，然后前往“存储管理”中查看 U 盘是否正常识别

打开软件“PuTTY”连接 K3 的 Telnet，“主机名称（或 IP 地址）”处填写路由器地址 192.168.2.1，“连接类型”选择”Telnet”，然后点击“打开”

输入命令“mount”查看 U 盘是否挂在成功，成功的话会看到 sda 开头的设备，我这边是显示 sda1

输入如下命令进行备份操作

备份完成后，进入“存储管理”中点击“安全移除”，如果无法移除，那就直接拔 U 盘，然后将备份的文件拷贝到电脑或者网盘进行二次备份。

第三部分

刷入官改固件

打开”Tftpd64.exe”,”Current Diectory”为路由器固件所在的文件夹，点击“Browes”可修改，一般默认既为 Tftpd64 所在文件夹的路径，“Server interfaces”为连接路由所获取到的本机电脑的 ip 地址，手动选择 ip 地址为“192.168.2.x”，x 为路由自动获取到的地址，可能不一样，我这边的 ip 地址为 192.168.2.100

再次打开“PuTTY”，如上次步骤输入 ip 地址为 192.168.2.1，连接类型为“Telnet”，然后单击“开始”进行连接，然后输入如下命令刷入。

上传成功后输入刷入固件命令

稍等片刻，会出现如下错误，rmmod dhd failed: No such file or directory，说明路由器已经断开了 PuTTY 的连接，K3 的屏幕上也显示为“系统升级中”，这时候等了 2,3 分钟进入 192.168.2.1 即可。

成功后打开 192.168.2.1 登录后台，查看“系统信息”，可以看到已经已经刷入成功，至此刷入完成
